# Uni-MAP-Assignments

This repository contains assignments for the course  
**Advanced Methods of Programming (MAP)**.

## Structure

- A2 – Basic statements and expressions
- A3 – Program state and execution stack
- A4 – File handling
- A5 – Heap memory and references
- A6 – Concurrency
- A7 – GUI integration

Each folder contains the full source code for the corresponding assignment.
